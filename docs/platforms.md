# Supported platforms

By default Afterburn uses the Ignition platform ID to detect the environment where it is running.

The following platforms are supported, with a different set of features available on each:

* aliyun
  - Attributes
  - SSH Keys
* aws
  - Attributes
  - SSH Keys
* azure
  - Attributes
  - Boot check-in
  - SSH Keys
* azurestack
  - Boot check-in
  - SSH Keys
* gcp
  - Attributes
  - SSH Keys
* openstack
  - Attributes
  - SSH Keys
* openstack-metadata
  - Attributes
  - SSH Keys
